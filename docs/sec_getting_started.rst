Installation
============

To install Shape-Link, use one of the following methods:
    
* from `PyPI <https://pypi.python.org/pypi/shapelink>`_:
    ``pip install shapelink``
* from `sources <https://github.com/ZellMechanik-Dresden/shapelink>`_:
    ``pip install .``
